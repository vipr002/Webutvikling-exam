Dokumentasjon på F1 API

Indroduksjon

På denne siden så skal vi vise hvordan du kan bruke vårt F1 API. All data du får tilbake er JASON.

Vi har tre hoved endepunkter:

  • http:localhost/xxxx/api/driver
  • http:localhost/xxxx/api/race
  • http:localhost/xxxx/api/team

Drivers


Karakter skjema

key Type Description
ID Int ID på driver
Name String Navnet på Driver
Age Int Alder på Driver
Nationality String Hvilke land Driver er fra.
Image String Bilde av Driver

Hent alle:

URL for å hente alle sjåfjører er: http://localhost:xxxx/api/driver

{"id":32,
"name":"Rudolph Blodstrupmoen",
"age":43,
"nationality":"Norsk",
"image":"rudolphblodstrup.jpg"}

Hent en sjåfør etter id:

Du kan få tak i en sjåfjør ved å legge til id på enden av linken som: http://localhost:xxxx/api/driver/1

Teams

Karakter skjema

key Type Description
ID Int ID på team
Manufacturer String Merke på bilen
Driver1 String Navnet på Driver1
Driver2 String Navnet på driver2
Image String Bilde av bilen

Hent alle:

URL for å hente alle teams er: http://localhost:xxxx/api/team

{"id":7,
"manufacturer":"Felgen AS",
"image":"iltempogigante.jpg",
"driver1":"Reodor Felgen",
"driver2":"Solan Gundersen"}

Hent et team etter id:

Du kan få tak i et team ved å legge til id på enden av linken som: http://localhost:5132/api/team/1

Races

Karakter skjema

key Type Description
ID Int ID på team
WinnerName String Navnet på vinneren
WinnerTime double Tiden til vinneren
GrandPrix String Navnet på landet de kjørte
NumberOfLaps int nummeret av runder de kjørte

Hent alle:

URL for å hente alle races er: http://localhost:5132/api/race

{"id":7,
"winnerName":"Reodor Felgen",
"winnerTime":92.68,
"grandPrix":"Zandvoort",
"numberOfLaps":45}

Hent et race etter id:

Du kan få tak i et race ved å legge til id på enden av linken som: http://localhost:5132/api/race/1
